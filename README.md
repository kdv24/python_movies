## Quick start

1.  **Clone the site.**

    ```sh
    git clone https://github.com/kdv24/python_movies

    ```

2.  **Start it up!**

    Start up the server and view it on your localhost.

    ```sh
    cd python_movies
    python3 app.py
    ```

    ```

    The site is now running at `http://localhost:5000`!
    ```

3.  **Troubleshooting**

    Make sure you are running python3.

    Run

    ```
    python3 --version
    ```
